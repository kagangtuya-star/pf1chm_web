游侠战斗流派 Combat Style
  游侠和其他可以获得游侠战斗流派职业能力的角色可以选择获得以下新的游侠战斗流派。角色在选择以下的游侠战斗流派后，可以选择获得该流派提供的奖励专长而无需满足其先决条件。

  狡诈（Deceptive）：如果游侠选择了狡诈流派，他在获得战斗流派专长能力时可以从如下专长中选择：虚招脱离（Disengaging Feint）UC，精通虚招，远程虚招（Ranged Fient）本书和双武虚招（Two-Weapon Feint）UC。在6级时，他可以将高等虚招和误导战术（Misdirection Tactics）本书加入专长列表；在6级时，他可以将回马枪（Disengaging Shot）UC和越位虚招（Passing Trick）UC本书加入专长列表。

  威慑（Menacing）：如果游侠选择了威慑流派，他在获得战斗流派专长能力时可以从如下专长中选择：炫目技巧，胁迫者（Enforcer）APG，血腥击杀（Gory Finish）UC和威逼。在6级时，他可以将粉碎防御和战士之威（Martial Dominace）本书加入专长列表；在6级时，他可以将惧栗屠杀（Dreadful Carnage）APG和精通重击加入专长列表。

  黑手（Underhanded）：如果游侠选择了黑手流派，他在获得战斗流派专长能力时可以从如下专长中选择，并且可以视为拥有13智力和寓守于攻作为先决条件：寓守于攻，闪避，精通阴招，精通盗取。在6级时，他可以将高等阴招和高等盗取加入专长列表；在6级时，他可以将快速阴招UC和快速盗取UC加入专长列表。



传码员 Code Runner（游侠变体）
  擅长迅速而准确的传递一段密码讯息，传码员可以带着自己的消息在跨越整个大陆，跋山涉水以后仍然完美无误地将密码带到目的地。传码员受过的特训让他不但拥有惊人的记忆力，即使是被捕受到拷打他也不会轻易让审讯者获得自己携带的秘密情报。


  本职技能：传码员将知识（本地）和语言学加入他的本职技能列表，但从他的本职技能中移除驯养动物和医疗。本能力调整了游侠的本职技能。


  天才记忆力 Mnemonic Genius（Ex）：传码员可以如同刻印般精确的记下与回忆情报，即使这些东西他完全看不懂是什么或者完全不理解也能硬背下来。如果花费一小时时间并通过一个DC15的语言学技能检定，传码员就可以背下一段最多150字的讯息，这条消息必须是他所能理解的语言。

  传码员的语言学检定结果每超过DC5点，他就能再记下额外的150字讯息。如果他需要背下的这些信息是加密的或者是用他不能理解的语言写成的，那么这个语言学检定DC+10。在同一时间内传码员最多只能记忆一段讯息。如果他记忆失败，那么传码员可以无限制重试记忆它，只是每次都要花费一小时时间。一旦传码员记录了一段讯息，那么在他花费头脑去记另一段讯息前他都可以准确无误的复诵他记录的这条消息

本能力取代理解动物。


  抵抗审讯 Resist Interrogation（Ex）：在4级时，传码员在对抗预言系效果时在豁免检定上获得相当于他的游侠等级一半的加值。在11级时，如果传码员成功的抵抗了一个可能会使施法者获得情报或知识的预言系效果的话，施法者不会知道他成功的抵抗了这个效果，而且传码员还可以同时透露给他假情报，这需要传码员进行一个唬骗检定来对抗施法者的察言观色检定。本能力取代猎手羁绊。


贵公子 Dandy（游侠变体）
  在大多数人把游侠的形象定格为脏兮兮坏脾气的巡林客时，贵公子则游走于文明世界的宫廷与宴会之间，以令人赞叹的礼节与仪态作为传递谣言和打入上流社会的伪装。虽然他也有一颗游侠之心，但贵公子的猎物是不胫而走的传言与秘密，他的猎场是豪华的舞厅与权贵的密室，他的手腕和那些荒野游侠的弓箭和长刀一样致命。


  本职技能：贵公子将唬骗，交涉，知识（本地），知识（贵族）和察言观色加入他的本职技能列表，但从他的本职技能列表中移除驯养动物，知识（地城），知识（地理），知识（自然）和生存。本能力调整了游侠的本职技能。


  国家宿敌 Favored Nation（Ex）：每当贵公子选择一个宿敌时，他都取而代之需要选择一个国家并研究其政治，经济和文化动态。在选择国家宿敌后，贵公子可以在对该国的贵族，官员，皇室与名流人物进行的唬骗，交涉，知识，察觉，察言观色和生存检定获得+2加值，除此之外他也像正常的宿敌能力一样对这些目标在攻击与伤害检定上获得+2加值。

  贵公子可以在未受训的情况下使用合适类型的知识检定来判断一个人物是否来自他的宿敌国家。在他提升等级时，贵公子会获得新的国家宿敌和国家宿敌加值成长，如同普通游侠获得新宿敌和宿敌加值成长一样。

  本能力取代宿敌。


  操纵流言 Rumor Empathy（Ex）：贵公子可以轻易操作某地的流言方向。本能力与理解动物职业能力相似，但贵公子需要花费一天（准确的说8小时）时间而非一分钟来操纵流言，而且他的交流目标是当地人的社交网络而非动物。

  成功的操纵流言检定可以让贵公子选择改变当地的流言内容，使它们对某人所持有的态度发生变化。贵公子可以让对于某人属于不关心状态的流言对释放出对他友善的讯号从而对此人产生积极影响，也可以让同样的流言对于此人的内容变得恶毒，产生负面的影响。

  操纵流言的难度取决于贵公子所在地方的人口规模。在一座村落或更小程度的定居点中这个DC是15，他想操纵流言的区域大小每从村落增加一级，他进行操纵流言的DC就增加5点，所需的时间增加两天。这个检定的DC上限为40和11天时间，足以让贵公子在一座大都市中操纵流言的传播。

  如果贵公子只想在一个较大地方中的特定区域（比如一座大都市中的精灵聚居社区），他可以用这个较小区域的实际规模来计算操纵流言的DC和所需时间。如果贵公子操作的流言涉及他本人，那么这个操纵流言的DC还要再增加10，因为他不可能让其他人不带偏见的在贵公子为人所知的时候客观地改变态度。本能力取代理解动物。


  偏好地形 Favored Terrain（Ex）：在3级时，贵公子必须选择城市作为他的偏好地形，但随着等级提升，他只会提升现有的偏好地形加值而不会获得新的偏好地形类型。在8级时，每当贵公子身处上流社会或是一次聚会中时，他的偏好地形加值额外+2。本能力调整了偏好地形。


  狐朋狗党 Hobnob（Ex）：在4级时，贵公子必须要选择他的盟友进行羁绊而不能选择动物伙伴。除了猎手羁绊的正常效果外之外，贵公子还可以在出席一次聚会或者活动时花费10分钟时间向重要的来宾介绍他的伙伴，让这些盟友可以获得贵公子的国家宿敌技能加值（不能获得伤害和攻击加值）。

本能力调整了猎手羁绊。


  贵公子魔法 Dandy Spell（Su）：在4级时，贵公子学习和其他大多数游侠不同的施法方法。贵公子使用他的魅力调整值而非感知调整值决定他的每日法术数量，法术DC和其他法术相关效果。贵公子可以自发施放吟游诗人法术列表中的法术（而非游侠法术列表），他使用等同于游侠等级的通灵者OA的每日法术数量和可知法术数量。本能力调整了游侠的法术。


  乱入 Party Crasher（Ex）：在7级时，贵公子可以找到乱入到上流人士聚会，宫廷或是其他需要足够多地位才能混进去的社交场合的方法。尝试乱入需要贵公子进行一个DC25的交涉技能检定并花费1d4小时去寻找合适的人。如果他的检定成功，这代表贵公子想办法弄到了一份参会邀请，支开了警卫或者是骗到了一个能把他带进该场合的女仆或贵妇。但是，本能力不能让贵公子乱入到小的社交圈子或是秘密集会中，他只能通过这个能力出现在公开的社交场合。

  本能力取代穿林。


帮派杀手 Guildbreaker（游侠变体）
  有些游侠精心训练自己的战技与打磨武器，但他们猎杀的对象并不是某种生物。这些老练的杀手专注于袭击并瓦解自己恨之入骨的某一非道组织。通过搜寻敌对帮派的成员，研究对象的活动模式和潜入卧底以刺探情报，帮派杀手可以以一人之力打垮整个盗贼公会。


  本职技能：帮派杀手将唬骗，易容和知识（本地）加入他的本职技能列表，但从他的本职技能列表中移除驯养动物，医疗和知识（地理）。本能力调整了游侠的本职技能。


  组织宿敌 Favored Organization（Ex）：每当帮派杀手选择一个宿敌时，他都取而代之需要选择一个组织（比如某个盗贼公会）并研究其行为模式和成员而非一种生物。帮派杀手的最初一级宿敌必须是组织宿敌，在此之后他升级获得新的宿敌加值时，他可以任意选择生物或是组织宿敌。在选择组织宿敌后，帮派杀手可以对任何该组织的合法成员（它的非成员盟友不行）获得宿敌能力的好处，无视其生物类型。帮派杀手必须知晓自己的目标确实属于他的组织宿敌才能获得本能力的好处

  本能力调整了宿敌。


  理解城市 Read the City（Ex）：帮派杀手的生活通常是在一座城市或者其他的大型人口聚居地度过的。作为经验的积累，帮派杀手可以直觉地追踪到在他身边出现的提示信息和合适的信号。当他身处城市环境中时，帮派杀手可以使用生存技能来搜集信息，但他只能通过这一能力了解到居住于该地的生物的活动情况（比如他们的种族，社会地位，职业，诸如此类）。因为他并不是通过和当地人交流而是通过寻找隐藏的信号来了解这些信息，帮派杀手不会让其他人知道他了解这些情报的目的。本能力取代理解动物。


  潜伏 Deep Cover（Ex，Su）：公会杀手知晓如何从内部破坏他作为目标的敌对组织，为此，他准备了足够的资源并留下数个以备万一的陷阱来增强自己伪装成该组织成员的隐蔽性的连环套。在4级时，帮派杀手在伪装成隶属于他的宿敌组织的成员时，他在所有的唬骗和易容检定上获得相当于组织宿敌加值一倍数量的环境加值（足够加值只能帮助他伪装一个属于该组织的成员身份，但是不能帮助他改变外貌或是伪装成该组织的一个特定成员）。

  当有一个属于该组织的成员试图看穿帮派杀手的伪装时，通过本能力获得的唬骗检定加值可以与帮派杀手通过组织宿敌能力获得的永久性唬骗加值叠加。当他伪装成自己的宿敌组织的成员时，任何尝试使帮派杀手说出真相的法术效果对他自动失败（如同他拥有法术花言巧语[Glibness]的好处一般），但他必须自称自己是属于该组织的成员才行。其他更为非直接的此类法术询问，如询问他是否属于某个非宗教性质的邪恶组织，或是询问他的信仰，则帮派杀手不会受到本能力的保护。本能力取代猎手羁绊。


  穿人 Crowd Stride（Ex）：在7级时，公会杀手可以轻松地穿越人潮耸动的拥挤街道，在穿越人群造成的阻碍时，公会杀手的移动不会受到影响。本能力取代穿林。


禁卫 Sentinel（游侠变体）
  禁卫是专业的守位，他们训练自己的感知，抓住一切试图绕过他们的不懈防卫的间谍和渗透者。城市卫队，大商人，甚至是犯罪集团都渴望一位禁卫为他们服务——无论是为了保卫政府力量的权威，又或是反其道而行，揪出组织中隐藏的所有自以为安全无虞的卧底。


  辨识专家 Mugshot（Sp）：禁卫可以选择花费1分钟时间，将一个目标的外貌牢牢记在脑海里（通过此人的相片，雕刻，有形的幻术成像也可以），并形成条件反射一般的完美记忆。从这以后，禁卫可以在从人群中辨识出此人，或是看破此人使用易容伪装时的察觉检定上获得+4加值。当易容后的该人出现在禁卫面前时，禁卫可以自动获得一次看穿他的易容的察觉检定机会，即使此人并没有引起他的注意也是一样。禁卫每次最多可以记住相当于他的感知调整值（最少为1）个人的外貌特征。如果禁卫在他记忆人的外貌数量达到上限以后还要再记下更多人，那么他必须选择忘记一个之前记住的人才能记住新的人。本能力取代理解动物。


  离奇机警 Uncanny Alertness（Ex）：在4级时，禁卫从不会在任何时候放松警惕而被人抓到可乘之机。在加入战斗前，禁卫不会陷入措手不及状态（但其他会造成措手不及的效果和能力仍然可以使他措手不及）。

  在7级时，禁卫总是可以在突袭轮时行动。如果他可以在突袭轮中行动或是战斗不存在突袭轮，那么他在先攻检定上获得相当于自己的游侠等级一半的加值。本能力取代猎手羁绊和穿林。


  入侵者警戒 Sense Intruder（Su）：在6级时，禁卫可以在站定凝神时以超自然的速度扩展自己的感知力，在原地建立起令精度人叹为观止的警戒区。在任何时候，禁卫都可以花费1分钟时间来使用本能力部署警戒位置。为了进行警戒，禁卫必须站立保持静止，除了扫瞄周遭环境外不能进行任何动作，包括交谈这样的自由动作。禁卫由此获得的超自然感知力可以包括他周围20尺半径的距离。任何尝试通过此区域中，体型不小于超小型（Tiny）的生物的存在会立刻被禁卫感觉到（本能力的这个部分修改是一个预言系法术效果但不能被任何阻碍这类效果的法术所隔断，除了极端强大的法术如法术心灵屏障[Mind Blank]）。除此之外，除非该生物拥有自动避免包括视觉，嗅觉和听觉察觉的效果（如同时处于法术隐形术与消除气味[Negate Aroma]APG的效果影响）保护，否则禁卫可以立刻以一个直觉动作通过一个察觉检定来定位该生物的位置，忽略该生物可能从任何通过阻碍观察者感知的法术，能力和效果（如法术隐形术或是使用伪装）处获得的潜行检定加值。

  然而，这种需要高度集中注意力的超自然感应对于身体的负荷是巨大的。在使用本能力时，禁卫视为在同样长度的时间内进行强行军旅行（Hustling Overland），只不过他不是从一个单独的地点出发而已。在激活入侵者警戒时，禁卫在第一个小时的警戒时不承受惩罚；如果他在一个睡眠周期内使用警戒达到第二个小时，他承受1点非致命伤害；在此之后他每多警戒一小时，禁卫就要承受比上一小时承受的多一倍的非致命伤害。只要他因为使用本能力而受到非致命伤害，禁卫就会陷入力竭状态。在计算禁卫一天内会因为使用本能力而承受多少非致命伤害时，他每小时保持入侵者警戒状态就视为进行了一小时强行军。

  在10级时，每当禁卫可以获得一个游侠战斗流派专长时，他可以取而代之，增加入侵者警戒的有效作用半径10尺。本能力取代游侠在6级时获得的游侠战斗流派并调整了游侠战斗流派能力。


  辨识狩猎 Mugshot Quarry（Ex）：在11级时，禁卫可以在他激活狩猎目标职业能力（以及19级的精通狩猎目标职业能量）时，将他使用辨识专家记忆的目标设为他的狩猎目标的对象，即使该生物不在他的视野范围中。本能力调整了狩猎目标和高等狩猎目标。


交通员 Transporter（游侠变体）
  只要你找到一个交通员，那么他就可以帮你顺畅无阻地来往于盘查重重的险境与疆界之间。逃离压迫政府的异见者，惊慌失措的难民，或是走投无路的逃亡罪犯都请求交通员的帮助。因此，大多数交通员属于截然不同的两派：心怀正义与善良，帮助政治犯和受苦者远离危险的守护天使，或是借自己的技艺和关系，从慌不择路的逃亡者手中榨取最后一个子儿的逐利之辈。


  无踪 Trailbreaker（Ex）：交通员会习惯性地抹除自己之前留下的痕迹。任何尝试追踪交通员或是他所从属的团队的的生存技能检定DC都会增加相当于他的游侠等级一半的数值（至少为1）。本能力提供的加值不会与其他类似效果的职业能力叠加。本能力取代理解动物。


  秘密路径 Plot Course（Ex）：在3级时，交通员可以花费1d4小时时间并进行一个DC15的知识（地理）检定来为自己安排一条从目前位置到达某个目的地的秘密行动路线。无论如何，交通员安排的秘密路径总是非直接的，因为它必须考虑偷渡的安全性。在此之后，交通员在他在秘密路径上旅行时，在先攻检定，知识（地理），察觉，潜行和生存检定上都可以获得+2加值。

  在8级时，如果交通员在这一知识（地理）技能检定上获得的结果超过DC10点或更多，那么他从本能力获得的加值提升到+4。在13级时，如果交通员在这一知识（地理）技能检定上获得的结果超过DC20点或更多，那么他从本能力获得的加值提升到+6。在13级时，如果交通员在这一知识（地理）技能检定上获得的结果超过DC30点或更多，那么他从本能力获得的加值提升到+8。

  交通员在同一时间内只能从一条秘密路径上获得好处。每当交通员开始尝试寻找另一条秘密路径，他就会立刻失去目前从现有的这条秘密路径上获得的好处。交通员可以重试设计秘密路径的知识（地理）检定，但每次都要花费1d4小时时间。本能力取代偏好地形。


  偷渡者羁绊 Smuggler`s Bond（Ex）：在6级时，交通员必须选择队友作为他的猎手羁绊而非动物伙伴。不过，通过一个移动动作，交通员并不能给予他的队友以自己的宿敌加值，而是持续性的给予他们自己从秘密路径上获得的加值。本能力取代猎手羁绊。


  藏匿处 Hideaway（Ex）：交通员可以花费1小时时间，尝试一个DC20的生存检定来建造一个隐蔽的藏匿处，避开敌对生物的视线检查和巡视。交通员建立的藏匿处可以容纳他的生存检定每超过DC2点/一个生物。这个藏匿处的建设加入了用来扰乱嗅觉探测的材料，因此无论从外还是从内它都会阻隔拥有灵敏嗅觉的生物使用它来探测；除此之外，藏匿所的地面使用了不同高度的地板或不同的材质建设，因此无论从外还是从内它都会阻隔拥有震颤感知的生物使用它来探测。最后，这个精巧的隐蔽设计可以让藏匿处外的生物在尝试发现内部有生物的察觉检定上承受-10减值，如同隔着一堵1尺厚的墙壁一样；但是，藏匿处内部的生物在向外观察时只在察觉检定上承受-2减值。

  这些好处在对一个异常熟悉藏匿处的环境的生物面前（比如他的家门口）无效。本能力取代穿林。
