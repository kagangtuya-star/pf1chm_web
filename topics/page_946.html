预设装备包 Equipment Packages

  考虑到PF现在已经出了一狗票物品，写卡的时候挑选合适的装备现在成了一件让人头疼的事情——对于那些需要写新角色卡的玩家来说尤其是如此。在以下章节，我们列出了数种可以帮助角色快速便捷地开展冒险所用的预设装备包任君挑选——无论职业为何，总会有一款合适你。以下列出的预设装备包中的物品搭配是固定，只有GM有权力对其中具体的物品做出调整（根据个别冒险者和冒险的需要，我们也建议GM对预设装备包进行适当微调）。

  选择新的背景元素：出门装的PC可以选择获得以下列出的预设装备包之一而不是他使用角色的起始金钱去购买装备。本背景元素是一个装备背景（Equipment Trait），装备背景元素和其他的背景元素（战斗，社交，信仰，魔法）的概念类似，但是首次出现于早起的玩家伴侣Pathfinder Player Companion Adventurer's Armory中被引入。当然，除了通过背景元素获得以外，玩家也可以正常的支付1000GP的市价来购买这些预设装备之一包，或是作为交换，用自己的忠实服务从富有的NPC手中获取一件预设装备包作为奖赏。


  出门装 Well-Provisioned Adventurer（装备）：你从年轻时就知道自己的未来将成为一个浪迹天涯的冒险者而不是庸庸碌碌的普通人。为了这个梦想，你一直努力社畜，零敲碎打地积攒起以后会用上的每件冒险装备。
  你可以选择获得下列的预设装备包中的一件。如果你在创建角色时选择了本背景元素，那么你将不会获得任何建立角色的起始金钱。

魔术大师包 Arcane Adept Package

  奥术魔法的研习者会在自己的行李中搭配各色各样的魔法物品来为成功的冒险旅程做准备。在队伍有碰到麻烦的时候，他总是能自信满满地拿出可以正好解决大家眼前困境的神奇道具。这套装备包适合的职业包括奥能师，术士，女巫或法师，但吟游诗人与召唤师也可以考虑入手这一套。事实上，魔战士也可以利用本装备包，但作为微调，请将所有的卷轴和魔杖置换为一件精制品武器。

  武器：轻型十字弓和10发弩矢，木棍

  战斗装备：2瓶强酸（Acid），2份通晓语言（Comprehend Languages）卷轴，侦测秘门（Detect Secret Doors）卷轴，闪光尘（Glitterdust）卷轴，2份鉴定术（Identify）卷轴，坐骑术（Mount）卷轴，魔绳术（Rope Trick）卷轴，暴雷石，法师护甲魔杖（剩余16发充能）

  其他装备：背包（Backpack），铺盖卷（Bedroll），腰包（Belt Pouch），5根蜡烛（Candle）燧石和钢片（Flint and Steel），墨水笔（Inkpen），浮游明焰（Ioun Torch）UE，日记本（Journal）UE，放大镜（Magnifying Glass），餐具（Mess Kit）UE，卷轴匣（Scroll Box）UE，50尺丝绳，法术材料包，5天份的干粮，水袋，3GP零钱

  全重：45磅（对于小体型的角色来说是28.75磅）


 

帅气勇者包 Daring Bravo Package


  作为一个称职的勇者，无论是社交还是战斗都应该毫不退缩，如鱼得水才对，而本装备包正好提供了足够的物品来让你轻松地适应这两重角色。为同时注重品味，眼球经济和战力的冒险者量身打造，这套装备包中的每一件装备都做工精美，应买家的要求而专为个人设计。本装备包适合的职业包括吟游诗人，战士，游荡者，游荡剑客或是侠客，不过一些骑士，调查员与圣武士可能也会发现它对自己潜在的价值。如果购买本装备包的角色是一个施法者，那么其中将会包含一只法术材料包。

  防具：精制品链甲衫

  武器：轻型十字弓和10发弩矢，精制品细剑，闷棍，炼银匕首

  战斗装备：2瓶强酸，炼金术士的恩惠（Alchemist`s Kindness）UE，2瓶治疗轻伤（Cure Light Wounds）药水，3支日光杖

  其他装备：背包，铺盖卷，腰包，一瓶好酒，5支粉笔（Chalk），礼服（Courtier`s Outfit）和价值50GP的珠宝首饰，扁壶（Flask），燧石和钢片，梳妆盒（Grooming Kit）UE，精制品乐器，餐具，镜子，香水或古龙水（Perfume or Cologne）UE，麻袋（Sack），玺戒（Signet Ring），50尺丝绳，5天份的干粮，水袋，磨刀石（Whetstone），5GP零钱

  全重：76.5磅（对于小体型的角色来说是40.5磅）


 

圣洁斗士包 Holy Warrior Package

  侍奉神的战士们运用自己的武力与狂热的信仰心与敌人抗争。即使信念坚定，战士们仍然了解自己必须要用到一些特殊的装备来克服魔物的邪力。本装备包适合的职业包括牧师，审判者，圣武士，战斗祭司，甚至是信仰虔诚的战士。总体来说，这套装备包中的物品可以帮助角色更好地成为支援整个团队战斗的辅助者。作为必要的调整，GM应当根据角色的信仰将这套装备包中的精制品长剑换为其他对应的精制品武器。如果购买本装备包的角色是一个施法者，那么其中将会包含一只法术材料包。

  防具：重钢盾，精制品胸甲

  武器：寒铁钉头锤，重型十字弓和10发弩矢，精制品长剑

  战斗装备：3瓶炽火胶（Alchemist`s Fire），4瓶圣水（Holy Water），祝福武器（Bless Weapon）油，2瓶治疗轻伤药水，防护邪恶（Protect from Evil）药水，3支日光杖

  其他装备：背包，铺盖卷，腰包，5根蜡烛，5支粉笔，撬棍（Crowbar），燧石和钢片，银质圣徽（Silver Holy Symbol），餐具，麻袋，50尺丝绳，5天份的干粮，水袋，7GP零钱

  全重：102磅（对于小体型的角色来说是57.25磅）


 

求闻学士包 Lore Seeker Package


  对于失落秘密的探索往往会将学者们引向远古而诡秘的遗迹，而本装备包则可以满足他们在这些环境中的冒险需要。这套装备包并没有包括太多的战斗武备，但它的内容物会让一个炼金术士，吟游诗人，武僧，术士或是法师满意的。如果购买本装备包的角色是一个施法者，那么其中将会包含一只法术材料包（以及一枚圣徽，如果有需要）。

  武器：轻型十字弓和10发弩矢，木棍，炼银匕首

  战斗装备：3瓶炽火胶，抗毒剂（Antitoxin），消抹术（Erase）油，两瓶治疗轻伤药水，防护邪恶药水，绊足包（Tanglefoot Bag），暴雷石

  其他装备：背包，铺盖卷，腰包，5根蜡烛，5支粉笔，指南针（Compass）UE，撬棒，永明火把（Everburning Torch），抓钩（Grappling Hook），燧石和铁片，2支墨水笔，2本日记本，放大镜，绘图师工具（Mapmaker`s Kit）UE，餐具，麻袋，信号哨（Signal Whistle），50尺丝绳，5天份的干粮，旅者万用工具（Traveler`s Any-tool）UE，水袋，8GP零钱

  全重：44磅（对于小体型的角色来说是27.75磅）


 

神意指引包 Mystic Guide Package


  本装备包的针对人群的各式各样的神术施法者，比如牧师，德鲁伊，先知还有审判者。运用装备包为他们提供的物品，这些神术使更为适合后排的辅助职责，用法术和能力增强他们在身前战斗的队友。

  防具：皮甲，轻木盾

  武器：短矛，投石索和10颗投石弹

  战斗装备：抗毒剂，祝福念珠（Blessing Bead，如同次等祈祷念珠项链 [Lesser Strand of Pray Beads]，但缺少其中的医疗念珠[Healing Bead]）UE，2瓶圣水，2份治疗轻伤卷轴，忍耐环境（Endure Elements）卷轴

  其他装备：背包，铺盖卷，腰包，5根蜡烛，5支粉笔，燧石和钢片，医疗包（Healer`s Kit），银质圣徽，浮游明焰，餐具，镜子，麻袋，50尺丝绳，嗅盐（Smelling Salt）UE，肥皂（Soap），法术材料包，5天份的干粮，水袋，4SP零钱

  全重：67磅（对于小体型的角色来说是39.25磅）


 

历险骑士包 Questing Knight Package


  除非他出身豪门贵族，天生不缺钱，不然一个骑士冒险者总是会需要攒下一整套行头才好踏上征途。这套装备包适合的职业包括骑士，战士和圣武士。如果购买本装备包的角色是一个施法者，那么其中将会包含一只法术材料包（以及一枚圣徽，如果有需要）；如果购买本装备包的角色本身就拥有坐骑职业能力，那么将本工具包中的战马和长枪调整为一支精制品长枪。

  防具：半身甲，重木盾

  武器：4支标枪，长枪，长剑

  战斗装备：3支日光杖

  其他装备：背包，旗帜，铺盖卷，腰包，扁壶，燧石和钢片，餐具，10尺长的长杆（Pole），麻袋，铲子（Shovel），玺戒，50尺丝绳，5天份的干粮，水袋，磨刀石，8GP零钱

  坐骑：经过战斗训练的重型战马（Heavy Horse），附带嚼子和缰绳（Bit and Bridle），军用座鞍（Military Saddle），鞍袋（Saddlebags）

  全重：127.5磅（对于小体型的角色来说则是73.75磅）


 

暗中观察包 Shadowy Stalker Package

  无论是在城市的暗街陋巷又或是地城的穹顶阴影下潜行，那些需要隐秘活动，一击得手而后远遁的冒险者会需要本装备包提供的支持。这套装备包适合的职业包括游侠，游荡者和杀手，而某些吟游诗人与调查员也可能会被它提供的帮助所吸引。如果购买本装备包的角色是一个施法者，那么其中将会包含一只法术材料包。

  防具：精制品皮甲

  武器：3把匕首，精制品镰刀，闷棍

  战斗装备：3瓶炽火胶，3根发烟棒（Smokestick），2支日光杖

  其他装备：铺盖卷，腰包，铁蒺藜（Caltrops），5根蜡烛，5支粉笔，易容工具包（Disguise Kit），燧石和钢片，玻璃刀（Glass Cutter）UE，精制品背包APG，精制品盗贼工具（Thieve`s Tools），餐具，麻袋，50尺丝绳，百衣套袖（Sleeves of Many Garments）UE，5天份的干粮，水袋，1GP零钱

  全重：63磅（对于小体型的角色来说则是33.25磅）


 

荒野浪客包 Wildness Wanderer Package

  本工具包适合那些轻装上阵，在野外驰骋战斗的冒险者，比如野蛮人，猎人或游侠。如果购买本装备包的角色是一个施法者，那么其中将会包含一只法术材料包以及施放自然法术所用的冬青与榭寄生。GM可以考虑根据角色的战斗风格来调整本装备包武器部分的精制品武器，比如一个擅长弓术的游侠可以选择拥有精制品复合长弓而不是精制品巨斧。

  防具：快速装卸轻木盾（Light Wooden Quickdraw Shield）UE，精制品镶嵌皮甲

  武器：寒铁连枷，复合长弓和20支箭矢，精制品巨斧，短剑

  战斗装备：3瓶炽火胶，抗毒剂，感官锐化（Keen Sense）APG药水，3支日光杖

  其他装备：背包，铺盖卷，腰包，5根粉笔，攀爬工具包（Climber`s Kit），寒带服装（Cold Weather Outfit），鱼钩（Fishhook），燧石和钢片，餐具，10尺长的长杆，铲子，信号哨，50尺丝绳，5天份的干粮，水袋，2SP零钱

  全重：108.5磅（对于小体型的角色来说则是63.25磅）

 


出自位面行者手册（Plane-Hopper's Handbook）的剩余三个预设装备包

蒙福卫士包Blessed Warden Package

蒙受祝福的信仰卫士们会准备好应对来自邪恶位面的阴险袭击的防护措施。这套装备包适合牧师、德鲁伊、审判者、先知和圣骑士使用，部分特定装备适合于那些准备将要前往深渊（混乱邪恶）位面的冒险者，但经过调整也可以适应其他邪恶位面，例如将防护混乱卷轴更换为防护秩序卷轴，就能设计出一套为地狱旅行者准备的装备包。

防具：精制链甲衫
武器：寒铁短矛，轻弩和20支弩矢
战斗装备：3瓶圣水，3张治疗轻伤卷轴，2张防护混乱卷轴，2张防护邪恶卷轴，移除诅咒卷轴
其他装备：背包，铺盖卷，5根蜡烛，5支粉笔，撬棍，浮游明焰，餐具，铲子，50尺丝绳，法术材料包，5天份的干粮，水袋，木质圣徽，13GP零钱
全重：74磅（对于小体型的角色来说是44.5磅）


痛击虚体包Corporeal Warrior Package

普通装备并不适合对付没有实体的敌人，而准备前往星界的战士们必须准备好如何应对潜藏在星界迷雾中无所不在的无形威胁。这个装备包适合战士和圣骑士使用，并且如果PC能够施放法术，在这套装备包中添加一个法术材料包。

防具：精制胸甲
武器：重弩和10支弩矢，重型硬头锤，精制长剑
战斗装备：炽火胶，法师护甲药水，护盾术药水，魔化武器油
其他装备：背包，铺盖卷，5根蜡烛，5支粉笔，撬棍，浮游明焰，餐具，铲子，50尺丝绳，5天份的干粮，水袋，10GP零钱
全重：87磅（对于小体型的角色来说是48.5磅）


位面旅者包Planar Traveler Package

这套装备包为位面旅行者们准备了对付在目标位面可能遭遇的挑战的措施，适合轻甲战斗人员如吟游诗人和盗贼使用。如果购买它的PC能够施放法术，在这套装备包中添加一个法术材料包。

防具：精制镶嵌皮甲，轻木盾
武器：复合长弓和20支箭矢，精制钉头锤
战斗装备：3份炽火胶，气泡术药水，2瓶治疗轻伤药水，2瓶忍受环境药水
其他装备：背包，铺盖卷，5根蜡烛，5支粉笔，撬棍，浮游明焰，餐具，铲子，50尺丝绳，5天份的干粮，水袋，11GP零钱
全重：75磅（对于小体型的角色来说是42.5磅）
