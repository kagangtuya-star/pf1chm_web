异能者（Psychic）
对于所有具有情感的存在来说，其意志中的力量都可以和伟大的魔法神器或者圣地相比拟。通过开启这座充满精神能量的宝库，异能者能够重塑自身周围的世界、他人的精神、甚至是打开通往其他位面的道路。对于异能者来说，没有什么秘密无法解开，没有任何地方遥不可及，她能够从任何类型的精神魔法（psychic magic）中汲取力量。异能者有许多可以拓展自身精神力的方法，而她们所遵循的条理和准则也会进一步影响自身的能力。

角色定位（Role）：具有大量法术的异能者能够处理各种情况，但是她们尤其擅长移动和操纵物体、外加读取和影响他人的思维。

阵营（Alignment）：任意

生命骰（Hit Die）：d6

起始财富（Starting Wealth）：2d6 x 10gp （平均 70gp）

本职技能
异能者的本职技能是：唬骗（魅力），工艺（智力），交涉（魅力），飞行（敏捷），威吓（魅力），知识(全部)（智力），语言学（智力），察觉（感知），专业（感知），察言观色（感知）以及法术辨识（智力）。
每级技能点数：2 + 智力调整值。

职业能力
后述内容为异能者的职业能力。

武器与防具擅长（Weapon and Armor Proficiency）：异能者擅长所有简易武器（simple weapons），但是不擅长任何种类的护甲或盾牌。

法术（Spells）：异能者能够施放异能者职业法术列表中的异能者法术（psychic spells）。她能够施放任何自身的已知法术，而这不需要事先进行准备。要想获知或者施放一个法术，异能者的智力属性必须至少达到10 + 法术等级。对抗异能者的法术时，豁免检定的困难等级（Difficulty Class）等同于10 + 法术等级 + 异能者的智力调整值。
每天，异能者在每个特定的法术等级上，只能施放特定数量的法术。她的基础每日法术数量如上表所属。此外，如果她具有较高的智力属性，也能够获得额外奖励的每日法术（详细见核心手册 17页表1-3）。
异能者仅能选择数量有限的法术。在最初，异能者的已知法术仅为四个0级法术以及两个1级法术，这些法术由异能者自行从异能者法术列表中选择。每当她获得新的异能者等级时，便会获知1个或更多个新法术，如上表：异能者已知法术所述。和异能者的每日法术数量不同，她的已知法术数量不受自身的智力属性所影响；上表中的已知法术数量是固定的。
在4级以及之后每个偶数等级（6级、8级等等），异能者能够选择获知一个新法术，来取代一个她已经了解的法术。实际上，异能者会失去原有的法术，并更换为新的法术。新法术的等级必须和被更换的法术等级相同，而且新法术的等级也有限定，必须至少比当前她能够从异能者职业法术列表中施放的最高等级法术低1级。在任意一个给定的等级中，异能者仅能更换单一一个法术，并且必须在该级获得新法术的同时决定是否更换法术。
异能者不需要事先准备自己的法术。她能够在任何时候施放自身已知的任何异能者法术（psychic spells），不过前提是在她还未用尽对应法术等级上的每日法术。

念力（Knacks）：异能者可以使用一些念力，或者0级法术，一如上表所述。异能者可以像施放其他法术一般施放这些法术，但是这么做不会消耗任何法术位，而且可以重复使用。使用其他法术位施放的念力（比如由于使用了超魔专长）仍旧会如常消耗法术位。

精神力池（Phrenic Pool，Su）：异能者具有一个由超自然精神能量组成的能量池，在施放异能者法术（psychic spells）时，异能者能够从精神力池中抽取精神力对法术加以操控。精神力池的最高上限为异能者等级的1/2 + 她的感知或魅力调整值，具体使用哪个调整值取决于异能者的精神境界（psychic discipline）。精神力池会在经过8个小时休息或冥想后的每个早晨恢复；这些小时不必连续。根据精神境界的不同，异能者可能具有其他向精神力池补充精神力的方法或情况。超过精神力池上限的精神力会被浪费掉。

精神增幅（Phrenic Amplifications）：异能者开发出一种被称为精神增幅的特殊技巧来强化施法。异能者可以在使用精神魔法施放一个法术的时候启动一个精神增幅，这会对法术的效果或施法过程进行调整。被施放的法术称为联结法术（linked spell）。每当异能者施放一个法术时，她仅能启动一个精神增幅，而且这么做是施放该法术的动作的一部分。除非精神增幅的说明中注明了仅能与某些特定种类的法术进行联结，否则异能者可以对任何异能者法术（psychic spell）使用任何已知的精神增幅。异能者会在1级获得一个精神增幅，从后述列表中进行选择。在3级以及之后的每4个等级，异能者都能够再学会一个新的精神增幅。每个精神增幅仅能被选择一次。一旦选定了精神增幅，便无法更改。精神增幅需要异能者从精神力池花费1点或更多点的精神力才能够生效。


精神境界（Psychic Discipline，Ex或Sp）：每名异能者都可以通过一种特殊的理念来强化自身的精神力量，比如进行严格缜密的研究或者让自己达到某种特别的心理状态。这种理念被称作她的精神境界。异能者会根据自身所选择的精神境界来获得额外的已知法术。精神境界的选择必须在1级时做出；一旦选定，便无法更改。每个精神境界都会赋予异能者数个境界之力（在1级、5级和13级获得），而且会赋予她额外的已知法术。此外，精神境界也决定了异能者的精神力池和精神增幅所使用的属性类别。对抗精神境界能力（psychic discipline ability）的豁免检定DC为10 + 异能者的等级的一半 + 异能者的智力调整值。
1级起，异能者能够根据自身的精神境界学会一个额外的法术。她会在4级以及之后的每2个等级再学会一个额外法术，直至18级时学会最后一个。这些法术不计入在表：异能者已知法术的数量中。通过精神境界知晓的法术无法在更高的等级时更换为其他不同的法术。

侦测思想（Detect Thoughts，Sp）：2级起，异能者能够以类法术能力每日使用一次侦测思想（Detect Thoughts）。她还可以花费任意一个未使用的1级或更高等级的法术位来施放侦测思想。若她这么做的话，便会使用消耗掉的法术位的等级来判断意志豁免DC。当异能者获得该能力时，如果已经知晓了侦测思想的话，那么她可以习得一个不同的1级法术来取代侦测思想，失去原本的法术并更换为新的法术。

心灵连线（Telepathic Bond，Sp）：9级起，异能者能够以类法术能力每日使用一次心灵连线（Telepathic Bond），使用她的异能者等级作为她的施法者等级。异能者不能将自身排除在以此方式建立的心灵连线之外。她还能够花费任意一个未使用的4级或更高等级的法术位来施放心灵连线。使用更高等级的法术位来施放心灵连线不会为异能者提供更多的好处，但是她可以以此方式应用会提升法术等级的超魔专长。举例说明：若异能者拥有法术延时（Extend Spell），那么她可以以整轮动作来使用5级法术位施放延时心灵连线。当已经知晓心灵连线的异能者获得该能力时，她能够习得一个不同的4级法术取代心灵连线，失去原本的法术并更换为新的法术。

高等精神增幅（Major Amplifications）：在11级以及之后的每4个等级，异能者都可以高等精神增幅中进行选择，以取代精神增幅列表中的选项。

心灵感应（Telepathy，Su）：17级起，异能者能够和100尺内任何会说语言的其他生物进行精神沟通。她可以把复数生物加入到同一次心灵感应中，但是在同一时间内与复数生物进行精神沟通就好像同时与多个人说话沟通一般困难。当已经知晓心灵感应（telepathy；见本书法术）的异能者获得该能力时，她能够习得一个不同的5级法术取代心灵感应，失去原本的法术并更换为新的法术。

重塑自我（Remade Self，Sp）：20级起，异能者会受到后述法术之一的影响，作为常驻（constant）的类法术能力；秘法视力（Arcane Sight）、飞行术（Fly）、加速术（Haste）、英雄气概（Heroism）、回避侦测（Nondetection）或巧言术（Tongues）。该能力在异能者达到20级时进行选择，且不能更改。若异能者处于这个类法术能力影响之下的话，每当她施放一个异能者法术（psychic spell）时，都可以从她已知的精神增幅或高等精神增幅中，选取一个在正常情况下花费1点精神力的精神增幅，然后在不消耗精神力的情况下联结到法术上。而且这个精神增幅不被计算在每个法术最高只能受到一个精神增幅影响的正常限制中。

 

 

可选天赋职业奖励（Favored Class Options）

 

　　矮人：在施放异能者的防护系法术（psychic abjuration spells）时，法术的施法者等级增加1/2级，不过该加值仅在判断法术持续时间时生效。

　　精灵：异能者的精神力池（phrenic pool）的总量增加1/3点。

　　侏儒：异能者的精神力池（phrenic pool）的总量增加1/3点。
　　半精灵：获得1/6个新精神增幅（Phrenic Amplifications）。

　　半兽人：在施放异能者职业的法术时，由于受到伤害而进行的专注检定获得+1加值。

　　半身人：在判断境界之力（discipline powers）的每日使用次数或轮数时，将异能者的魅力加值视为比正常高1/3点。
　　人类：从异能者法术列表中获得一个已知法术。该法术的环级必须比异能者当前能够施放的最高法术环级至少低1级。
